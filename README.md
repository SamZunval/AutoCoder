# AutoCoder
This repository contains the AutoCoder project developed as part of the HyperSkill 'AutoCoder' project. 
A GitHub composite action that automates code genration using ChatGPT based on issue descriptions. When an issue with a specifed lable is opened, reopened, or labeled, this action wil run a script that sends the issue contents to ChatGPT, which will generate code, commit the generated files, and create a pull request automatically. This allows any user of this project to automate code productions leveraging  ChatGPT's ability to generate code for any use.
